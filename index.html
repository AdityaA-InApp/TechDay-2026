The prompting toolbox

01

Zero-shot

Just ask. The baseline — a clear instruction, no examples.

02

Few-shot

Show two or three worked examples; the model copies the pattern.

03

Chain-of-Thought

“Reason step by step” — accuracy jumps on anything with logic in it.

04

Role prompting

Set a persona and an audience to pin tone, depth, and vocabulary.

05

ReAct

Interleave reasoning with actions — think, act, observe, repeat.

😲 WHAT SURPRISED US

The agent handled an edge case we never discussed — because the acceptance criteria implied it. The spec was smarter than the meeting.

💥 WHAT BROKE

One ambiguous requirement produced confidently wrong code — three times in a row. The bug wasn't in the model. The bug was in the spec.

🚫 WHAT WE'D NEVER DO AGAIN

Leave "obvious" things unwritten. If it's not in the spec, it does not exist.
